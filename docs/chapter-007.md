
## Chapter 007. What is Reactive Programming?

* what is reactvie programming?
    * Reactive Programming is a new programming paradigm
    * asynchronous and non blocking
    * data flow as an event/message driven stream
    * functional style code
    * support backpressure on data stream
        * too many data from source
    * use when there is need to build and support high load with the available resources

* reactive app architecture
    * handle reqeust using non blocking style
    * netty is a non blocking server uses event loop model
    * using project reactor for writing non blocking code
